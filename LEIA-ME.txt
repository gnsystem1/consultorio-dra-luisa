COMO USAR

1. Extraia este arquivo ZIP.
2. Suba a pasta no Netlify.
3. Abra o link no Chrome.
4. Clique em instalar app.

IMPORTANTE
- Este pacote já está corrigido para NÃO cachear os dados da API do Google Apps Script.
- Se você atualizar no Netlify, desinstale o app antigo e instale novamente.
- Se ainda aparecer dado antigo, pressione Ctrl + Shift + R e limpe os dados do site no Chrome.
